pub async fn append<XBuf, X, XE>( &mut self, record: Record<M, Idx, X> ) -> Result<Idx, SegmentOpError<S, SERP>>
where XBuf: Deref<Target = [u8]>, X: Stream<Item = Result<XBuf, XE>> + Unpin,

Appends a new Record to this Segment.

Serializes the record metadata and bytes and writes them to the backing Store. Also makes an IndexRecord entry in the underlying Index to keep track of the Record.

Returns the index at which the Record was written.

Errors out with a SegmentError when necessary. Refer to SegmentError for more info about error situations and types.
