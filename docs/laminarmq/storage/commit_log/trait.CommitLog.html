Expand description

An abstract, append-only, ordered sequence of Record instances.

This trait acts as a generalized storage mechanism for storing our records. All our message-queue server APIs can be expressed using this trait.

Appends the given Record at the end of this CommitLog.

The Record may contain a stream of byte slices. Implementations are to exhaustively read the stream and append the corresponding byte slices as a single record.

Returns the index at which the Record was appended.
