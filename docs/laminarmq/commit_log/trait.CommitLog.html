Expand description

An ordered sequential collection of Record instances.

A Record in an CommitLog is addressed with an unique u64 offset, which denotes it position in the collection of records. The offsets in a CommitLog are monotonically increasing. A higher offset denotes that the Record appears later in the CommitLog.

This is useful for storing a sequence of events or operations, which may be used to restore a system to it’s previous or current state.
